a标签介绍： href: 1、url： 填写网页地址，可以起到连接、跳转的作用 2、download： 下载作用，可以放入文件、压缩包 3、 #id： 锚点，放入 id 属性，可以在当前页面进行跳转 4、#： 当href属性什么都没有的时候，通常放个#，代表当前网页 target: "_self" 在当前页面打开 "_blank: 新页面打开 如果标签比较多，需要在头部设置默认属性，参照head部分 伪类选择器:伪类用于向被选中元素添加特殊的效果。（元素在特定情况下才具备的。） 参照head去设置标签的伪类(不只是支持a标签，div和其他标签也支持) link 未访问（默认） hover 鼠标悬停（鼠标划过） active 链接激活（鼠标按下） visited 访问过后（点击过后） 四个伪类的顺序： link visited hover active （love hate 记忆方法！）
